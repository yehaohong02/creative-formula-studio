# Creative Formula Studio

AI买量视频创意公式工坊 - 基于 Coze API 的短视频脚本生成器

## 功能

- 3种爆款创意公式：玩法演示型、KOL解说型、真实模拟型
- AI 自动生成完整分镜脚本
- 支持自定义项目信息

## 部署

### Vercel 一键部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

### 本地开发

```bash
npm install
npm run dev
```

## 环境变量

复制 `.env.local.example` 为 `.env.local` 并填写：

- `COZE_API_KEY` - Coze 个人访问令牌
- `COZE_BOT_ID` - Coze Bot ID

## 技术栈

- Next.js 14 + TypeScript
- Tailwind CSS
- Coze API
